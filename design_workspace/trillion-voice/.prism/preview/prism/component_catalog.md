# Component Catalog

Compose from these. Only libraries listed here have working CLIs — do not reach for Aceternity/Reactbits (no CLI, not bundled).

## shadcn/ui

| Component | Use for | Install |
| --- | --- | --- |
| `button` | primary/secondary/ghost actions | `npx shadcn@latest add button` |
| `card` | content containers, product panels | `npx shadcn@latest add card` |
| `dialog` | modals | `npx shadcn@latest add dialog` |
| `sheet` | side panels / drawers | `npx shadcn@latest add sheet` |
| `tabs` | segmented views | `npx shadcn@latest add tabs` |
| `input` | text fields | `npx shadcn@latest add input` |
| `badge` | status pills, tags | `npx shadcn@latest add badge` |
| `tooltip` | hover-reveal marginalia | `npx shadcn@latest add tooltip` |
| `separator` | hairline dividers | `npx shadcn@latest add separator` |
| `scroll-area` | scrollable readouts/transcripts | `npx shadcn@latest add scroll-area` |
| `avatar` | identity chips | `npx shadcn@latest add avatar` |

## MagicUI

| Component | Use for | Install |
| --- | --- | --- |
| `grid-pattern` | ambient background grid (opacity >= 0.4) | `npx magicui-cli add grid-pattern` |
| `dot-pattern` | ambient background dots | `npx magicui-cli add dot-pattern` |
| `particles` | drifting particle field | `npx magicui-cli add particles` |
| `border-beam` | animated beam around CTA/cards on hover | `npx magicui-cli add border-beam` |
| `blur-fade` | staggered entrance for wordmark/blocks | `npx magicui-cli add blur-fade` |
| `text-reveal` | scroll/enter text reveal | `npx magicui-cli add text-reveal` |
| `number-ticker` | re-rolling numeric readouts | `npx magicui-cli add number-ticker` |
| `marquee` | scrolling logo/label strips | `npx magicui-cli add marquee` |
| `bento-grid` | feature bento layouts | `npx magicui-cli add bento-grid` |
| `animated-list` | streaming list of events/messages | `npx magicui-cli add animated-list` |
| `orbiting-circles` | orbiting nodes / constellation | `npx magicui-cli add orbiting-circles` |
| `shimmer-button` | premium CTA | `npx magicui-cli add shimmer-button` |
